# Resume-Tex
Resume built with Latex

#### Compilation Format
LuaLatex / pdfLatex

# Compilation Guide

## Linux/MAC OS
#### Requirements
1. Node JS
2. pdflatex

#### Compile using
`npm run build`



## Windows (Download MikTex to edit)
https://miktex.org/download

## Online Editor
https://www.overleaf.com/login
(Download the zip archive and import directly to overleaf projects)
